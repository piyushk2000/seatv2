## Setup & Run

### 1. Start Backend
```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```
Backend runs on: `http://localhost:8000`

### 2. Start Frontend
```bash
cd frontend
npm i
npm run dev
```
Frontend runs on: `http://localhost:5173`
